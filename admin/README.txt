====================================================
Gloria Trading 車両管理画面 - 使い方
====================================================

【アクセス方法】
ブラウザで以下のURLを開いてください：
  https://あなたのドメイン/admin/

【パスワードの設定方法】
  次のいずれかで設定します（GitHub には保存しません）。

  1) サーバー上の admin/password.txt にパスワードを1行で保存
  2) admin/.htaccess の SetEnv 行のコメントを外して設定
     SetEnv GLORIA_ADMIN_PASSWORD あなたの新しいパスワード

  ※ HTTP Basic 認証（.htpasswd）は使用しません。
     以前の Basic 認証設定が残っていると 500 エラー（白画面）になります。

【できること】
  1. 車両の追加・編集・削除
  2. 画像のアップロード（複数枚対応）
  3. 画像の削除・並び替え
  4. 保存すると自動的にサイトに反映

【注意事項】
  - images/vehicles/ フォルダに書き込み権限が必要です
    (パーミッション: 755 または 777)
  - data/vehicles.json に書き込み権限が必要です
    (パーミッション: 644 または 666)
  - さくらのレンタルサーバーの場合、FTPで
    上記ファイルのパーミッションを確認してください

【ファイル構成】
  admin/
    index.php  - 車両一覧・ログイン画面
    edit.php   - 車両追加・編集画面
    .htaccess  - セキュリティ設定・パスワード環境変数
